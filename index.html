Bitcoin-to-Dollar Converter

What is it?:
Here's an easy and quick way to dynamically display the current value of bitcoins. Simply create a SPAN tag and give the tag a data attribute of "data-bitcoin-value", and give that attribute a value of the number of bitcoins you'd like to convert to USD.

Syntax:
<span data-bitcoin-value='100'></span>

How it works:
Using jQuery's $.getJason function, the current value of a bitcoin is read from MtGox's ticker_fast API. Once the bitcoin value is read, all instances of <span data-bitcoin-value='100'></span> will then be injected with the caluclated value of the bitcoin. It's entirely done dynamically.

Hot to use it:

  1. Call jQuery
  2. Call bitcoin.js
  3. Create the span tag using the syntax above.

Who can use this:
This plugin was created with bloggers in mind. Instead of hardcoding the USD value of a bitcoin at the time the article was written, bloggers can now dynamically display the current value of bitcoins by adding a simple span tag within their article. See examples below on how those articles could have used this plugin.
